API¶

This part of the documentation covers all the interfaces of planets.

Coordinate Objects¶

class planets.Coordinate(x: int, y: int)¶

A class for keeping track of coordinates within the game.

Planet Objects¶

class planets.Planet(coordinate: Coordinate, planet_size: PlanetSize, home_player: Player | None = None, scheduler: SchedulerBase = None)¶

A class representing a planet in the game.

set_owner(player: Player) → None¶

Set the owner of the planet.

subscribe(observer: any) → None¶

Subscribe to events of the planet. The observer can be either an Observer or a function. It will be called and passed the planet object whenever the planet changes states.

PlanetSize Objects¶

class planets.PlanetSize(value, names=None, *, module=None, qualname=None, type=None, start=1, boundary=None)¶

An enum representing the size of a planet.

Planets Objects¶

class planets.Planets(tableau: Tableau, players: list[Player], number_of_planets: int = 10, min_distance: int = 5, scheduler: SchedulerBase = None)¶

A class for creating and keeping track of all the planets in the game. This is intended to be the core state machine for the game.

create_planet(planet_size: PlanetSize | list[PlanetSize] = [PlanetSize.SMALL, PlanetSize.MEDIUM, PlanetSize.LARGE], home_player: Player | None = None) → None¶

Create a planet within the game.

get_all_planets() → list[Planet]¶

Return a list of all the planets in the game.

get_planet_by_id(planet_id: uuid4) → Planet¶

Return the planet with the given planet’s id.

Player Objects¶

class planets.Player(name: str, color: tuple[int, int, int])¶

A class representing a player in the game.

set_home_planet(planet: Planet)¶

Set the home planet for the player.

Tableau Objects¶

class planets.Tableau(bottom_right: Coordinate)¶

A class represeting the game board. The upper left coordinate is always (0, 0). When creating a new Tableau, only the bottom right coordinate needs to be specified. This effectively sets the playing space.

get_size() → tuple[int, int]¶

Returns a tuple of the size of the tableau. Helpful for passing to pygame.display.set_mode().
